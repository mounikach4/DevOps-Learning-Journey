# 🐧 Day 3 - cat, awk & Linux File System

## 🎯 Objective
Practice the cat command, learn basic awk operations, and explore the Linux File System.

## 💻 Commands Practiced

### 📄 cat
- cat file.txt
- cat > file.txt
- cat >> file.txt
- cat -n file.txt
- cat -b file.txt
- cat -E file.txt
- cat -A file.txt

### 📝 awk

- awk '{print $1}' file.txt
- awk '{print $NF}' file.txt
- awk '$2>750000' file.txt

### 📂 Linux File System

- ls /
- ls /etc
- ls /var/log
- cat /proc/cpuinfo
- cat /proc/meminfo

## ✅ What I Practiced

- File creation and reading using cat
- Line numbering and hidden characters
- Basic text processing with awk
- Explored Linux File System directories
- Viewed CPU and memory information

## 💡 Key Learnings

- cat is used to create, read, and append files.
- awk helps process and filter text.
- Linux stores system files in a well-defined directory structure.
