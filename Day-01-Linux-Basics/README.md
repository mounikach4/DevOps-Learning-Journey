# Day 1 - Linux Basics (Commands & File System)

## 📖 What is Linux?

Linux is an open-source operating system widely used in servers, cloud computing, DevOps, networking, and embedded systems.

## 📖 Why Linux?

- Open Source
- Secure
- Stable
- Fast
- Multi-user
- Used in AWS
- Used in Docker & Kubernetes

## 📂 Linux File System

| Directory | Purpose |
|-----------|---------|
| / | Root Directory |
| /home | User Home |
| /etc | Configuration Files |
| /var | Logs |
| /tmp | Temporary Files |
