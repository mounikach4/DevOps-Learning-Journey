# 🐧 Day 2 - Advanced Linux Commands

## 🎯 Objective

Learn advanced Linux commands to retrieve system information, download files from the internet, search text inside files, use pipes, view file contents, and check command history.

## 📘 Topics Learned
### 🖥️ System Information Commands

- uname – Displays the operating system name.
- uname -a – Displays complete system information.
- uname -r – Displays the Linux kernel version.
- uname -m – Displays the machine architecture.
- uname -n – Displays the system hostname.

### 🌐 Web & Networking Commands

- wget – Downloads files from the internet.
- curl – Retrieves data from a website or server.

### 🔍 Text Searching

- grep – Searches for a word inside a file.
- grep -i – Ignores uppercase and lowercase while searching.
- grep -n – Displays matching lines with line numbers.

### 🔗 Pipe Operator

- | (Pipe) – Sends the output of one command as the input to another command.
Example: cat file.txt | grep Linux

### 📄 Viewing File Contents

- head – Displays the first 10 lines of a file.
- head -2 – Displays the first 2 lines.
- tail – Displays the last 10 lines of a file.
- tail -3 – Displays the last 3 lines.

### 📜 History Commands

- history – Displays previously executed commands.
- clear – Clears the terminal screen.
